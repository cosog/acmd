### 6.1.1 输入文本
#### 6.1.1.1 输入参数说明

表6-1 输入参数说明表

| **代码**                    | **名称** | **单位** | **类型** | **必填** | **备注** |
|-----------------------------|----------|----------|----------|----------|----------|
| **WellboreTrajectory**      | **井身轨迹**|          |          |          |          |
| MeasuringDepth              | 测量深度 | m        | []float64  | *       |          |
| DeviationAngle              | 井斜角   | 度       | []float64  | *       |          |
| AzimuthAngle                | 方位角   | 度       | []float64  | *       |          |

表6-2 输出参数说明表

| **代码**                         | **名称**       | **单位** | **类型** | **备注**                                   |

## 6.2 采集处理

#### 6.2.1.1 输入参数说明
